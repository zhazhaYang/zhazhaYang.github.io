GitHub+Hexo 搭建个人博客

发表于 2019-06-30 更新于 2019-07-14 分类于 搭建博客 评论数： 阅读次数：

今天刚搭建好自己的个人博客，把搭建的过程记录下来，作为自己的第一篇博客，或许也可以帮到需要的人。
我的系统是windows10
第一步 前期的准备
1、注册GitHub账号（没有的话）
2、安装Node.js 官网下载： [http://nodejs.cn/download/]
3、安装Git 官网下载：[https://git-scm.com/downloads]
cmd命令行检测安装和配置是否成功
注：cmd命令行调用git命令需要，git安装时选择use git from windows command或者配置电脑的环境变量Path

检测git安装是否成功的另一方法：在电脑上右键出现如下图

第二步 在本地新建Hexo项目
1、命令行新建并进入文件夹
2、安装hexo并检测hexo是否成功安装

3、在当前文件夹新建网站

4、在项目中安装npm模块

5、生成静态文件

6、开启本地服务,网站就在你本机上运行起来了，方便测试

注：发现占用可重设端口号：hexo server -p 端口号
然后可以用浏览器打开该地址查看结果。

第三步 关联GitHub Pages
1、GitHub上面新建repository
repository name：你的账户名+github.io
勾选initialize this repository with a readme

然后进入到刚新建的代码库的settings里面的GitHub Pages看到

2、联系hexo与GitHub Pages
(1)在第二步新建的本地项目文件夹中右键，点击Git Bash here
(2)初次运行git要配置用户信息，每一个 Git 的提交都会使用这些信息，并且它会写入到你的每一次提交中，不可更改；

(3)用git bash命令行工具进入.ssh文件，并使用下图的命令，再按三次回车键生成rsa密钥
注：第一次回车在默认文件id_rsa上生成ssh key；第二次，表示不输入密码；第三次，确认密码

(4)打开计算机用户目录下的.ssh文件夹里的id_rsa.pub，复制里面的内容
(5)进入你的GitHub
点击用户头像
选择Settings选项
再选择SSH and GPG Keys
点击New SSH key
填入title
粘贴刚才的复制到key里面，如下图
然后点击添加

(8)用cmd命令行在你的hexo项目中安装扩展

第三步 新建文章
1、用cmd命令行在你的hexo项目中
注：Hello是文章名

2、在项目中的source文件夹中的_posts文件夹中找到Hello.md文件，在里面添加内容

3、用cmd命令行在你的hexo项目中输入命令：hexo g -d 生成静态文件public并立即部署网站
4、打开https://用户名.github.io查看结果

然后就成功搭建基本的博客啦！
