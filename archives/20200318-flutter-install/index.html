Mac OS配置flutter开发环境

发表于 2020-03-18 分类于 flutter 评论数： 阅读次数：

系统要求

To install and run Flutter, your development environment must meet these minimum requirements:
Operating Systems: macOS (64-bit)
Disk Space: 2.8 GB (does not include disk space for IDE/tools).
Tools: Flutter depends on these command-line tools being available in your environment. bash、curl、git 2.x、mkdir、rm、unzip、which、zip

我的系统为macOS Majave 10.14.6

下载

官网：[https://flutter.dev/docs/get-started/install/macos]
下载后解压到你想放置的目录

添加到环境变量

打开命令行工具

在最下方添加flutter/bin的路径：
export PATH=/Users/yang/flutter/bin:$PATH
再生效配置并且检测是否成功

检测依赖的环境

检测后根据其提示按装所需要的工具

配置桌面应用的开发

从flutter 1.13开始，其作为alpha版本提供对macOS桌面的支持
官网链接:[https://flutter.dev/desktop]
1、检测可用channels

Flutter channels:

  • master
    dev
    beta
    stable
    2、切换channel到master, 并更新flutter sdk和依赖包
    注：stable是稳定的分支，官方建议的
    master是开发分支，稳定性低许多

3、开启macOS桌面支持

4、测试
