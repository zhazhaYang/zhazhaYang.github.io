hexo设置主题NexT

发表于 2019-07-02 更新于 2019-07-14 分类于 搭建博客 评论数： 阅读次数：

第一步：下载Next主题
在hexo项目的目录下，打开Git Bash here
输入：git clone https://github.com/iissnan/hexo-theme-next themes/next
注：themes/next 为在项目中存放的文件夹
第二步：打开站点的配置文件，找到theme字段，修改为theme: next如下

第三步：检测主题,查看主题是否替换成功

清除hexo缓存
开启服务器
开启调试模式（在终端显示调试信息）
结果大概是这样
hexo clean 
hexo s 
hexo --debug

第四步：在主题配置文件选择Scheme为Pisces如下，该配置路径为
themes/next/_configyml
注：可以自由切换，选自己喜欢的

第五步：在站点配置文件设置语言为zh-CN(简体中文)如下
（注意区分站点配置文件和主题配置文件）

第六步：开启自己需要的菜单
1、在主题配置文件的menu字段
注：这里的名字不要改，这是索引来的，改名字在language/zh-CN.yml

2、然后更改自己想要的名称

3、新建页面，以tags为例，其他一样

4、新建的页面可以在index.md文件内编辑内容
注：tags和catogories页面都需要加上type属性如下

第七步：设置footer（网站底部的栏目）
1、在站点配置文件的author字段可以输入作者姓名
2、在主题配置文件的powered字段和theme字段的enable都设为false，可以撤掉官方的版本说明
3、添加不蒜子访问量统计，在主题配置文件中的busuanzi_count字段的enable设置为true
注：在本地测试时显示的数据是不准确的

然后基本的NextT主题就配置好啦！如果还不满意，可以参考NextT文档，即下面的链接！
