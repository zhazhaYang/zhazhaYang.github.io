hexo博客添加到谷歌收录

发表于 2019-07-04 更新于 2019-07-14 分类于 搭建博客 评论数：

第一步：到谷歌搜索控制台验证网站所有权
1、打开GOOGLE的Search Console: https://search.google.com/
2、以网址前缀的资源类型填入你的域名

3、选择验证所有权的方式，个人域名采用域名提供商的方式，github.io则选用HTML上传或HTML标记。
Search Console验证网站所有权帮助：https://support.google.com/webmasters/answer/9008080?hl=zh-Hans
注：这里以个人域名为例，其他方式的话，下面的45小点不用做啦！

4、复制txt里面的记录值

5、打开个人的域名提供商的DNS解析控制平台
注：这里以啊里云为例
记录类型选择TXT
把刚才的复制粘贴到记录值

6、回到Google Search Console点击验证

第二步：在项目中生成sitemap
1、在项目的根目录下使用npm命令

sitemap:
path: sitemap.xml

hexo clean
hexo g -d

查看结果，可以看到public目录下生成了sitemap文件

第三步：提交站点地图到 google search console
找到google search console中的站点地图
填入你的域名+/sitemap.xml点击提交
{% asset_img add_sitemap.png %}
成功如下
{% asset_img success_add.png %}

不久之后，可以在谷歌搜索输入```site: 博客的域名``查看是否成功收录

发表于 2019-06-30 更新于 2019-07-14 分类于 搭建博客 评论数：

今天刚搭建好自己的个人博客，把搭建的过程记录下来，作为自己的第一篇博客，或许也可以帮到需要的人。
我的系统是windows10
第一步 前期的准备
1、注册GitHub账号（没有的话）
2、安装Node.js 官网下载： [http://nodejs.cn/download/]
3、安装Git 官网下载：[https://git-scm.com/downloads]
cmd命令行检测安装和配置是否成功
注：cmd命令行调用git命令需要，git安装时选择use git from windows command或者配置电脑的环境变量Path

检测git安装是否成功的另一方法：在电脑上右键出现如下图

第二步 在本地新建Hexo项目
1、命令行新建并进入文件夹
2、安装hexo并检测hexo是否成功安装
命令如下

3、在当前文件夹新建网站

4、在项目中安装npm模块

5、生成静态文件

6、开启本地服务,网站就在你本机上运行起来了，方便测试

注：发现占用可重设端口号：hexo server -p 端口号
然后可以用浏览器打开该地址查看结果。

第三步 关联GitHub Pages
1、GitHub上面新建repository
repository name：你的账户名+github.io
勾选initialize this repository with a readme

然后进入到刚新建的代码库的settings里面的GitHub Pages看到

2、联系hexo与GitHub Pages
(1)在第二步新建的本地项目文件夹中右键，点击Git Bash here
(2)初次运行git要配置用户信息，每一个 Git 的提交都会使用这些信息，并且它会写入到你的每一次提交中，不可更改；

(4)打开计算机用户目录下的.ssh文件夹里的id_rsa.pub，复制里面的内容
(5)进入你的GitHub
点击用户头像
选择Settings选项
再选择SSH and GPG Keys
点击New SSH key
填入title
粘贴刚才的复制到key里面，如下图

1、用cmd命令行在你的hexo项目中

2、在项目中的source文件夹中的_posts文件夹中找到Hello.md文件，在里面添加内容
